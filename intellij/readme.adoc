# Intellij IDEA

Intellij IDEAを初めて使うEclipser(Javaの人)を対象に説明する。

## 起動オプション

- 設定ファイル(eclipse.iniのようなもの)へJVMオプションを追加できる。
* [blue]_Windows_
** インストールフォルダ/bin/idea64.exe.vmoptions
* [red]_Mac_
** インストールアプリ/Contents/bin/idea.vmoptions
